## k0s kubeconfig admin

Manage user access

### Synopsis

Command dumps admin kubeconfig.

```
k0s kubeconfig admin [command] [flags]
```

### Examples

```
	$ k0s kubeconfig admin > kubeconfig
	$ export KUBECONFIG=kubeconfig
	$ kubectl get nodes
```
